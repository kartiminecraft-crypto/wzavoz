# WZAVOZ — 3D Model Showcase

Interactive 3D model showcase built with React, Three.js, and Vite.

**[View Live](https://0j5ahpwu.github.io/wzavoz/)**

## Tech Stack

- **React 18** + TypeScript
- **Three.js** + @react-three/fiber
- **Tailwind CSS v4**
- **Vite 6**

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
